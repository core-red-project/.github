## 🧪 What this is ✦

Internal infrastructure. Low-level tooling. Technical experiments  
that were never meant to be released — and eventually are ✦

Not end-user software.  
The backstage that makes the rest of the ecosystem possible (たぶん)

This branch is for whoever looks at the edges, not the center.  
Systems that don't need visibility to be solid.

## 🧠 The Stack ✦

<p align="center">
  <img src="https://skillicons.dev/icons?i=rust,go,ts,deno,python&theme=dark" />
</p>

<details>
<summary>✦ What's in the stack and why</summary>

<br>

**Rust** — the primary language. correctness, performance, zero compromises  
**Go** — concurrent systems and security tooling  
**TypeScript / Deno** — lightweight infrastructure and self-hosted dashboards  
**Python** — ML utilities and declarative training pipelines  
**n8n** — orchestration for intelligent automation that stays on-device

</details>

Low-level first.  
No unnecessary dependencies.  
Everything earns its place.
